# BrightHub Train Data - Comprehensive Codebase Overview

## Executive Summary

This is a Next.js 14 application with TypeScript, React 18, and Supabase for database management. It manages documents, training data, and AI-generated conversations with a three-step workflow for categorization and tagging.

**Tech Stack:**
- Frontend: Next.js 14, React 18, TypeScript
- Database: Supabase (PostgreSQL)
- API: Next.js API Routes
- UI: Radix UI, Tailwind CSS
- State: Zustand, React Query
- AI: Anthropic Claude API
